# canvas-particles

## `vectorField`

Generate Vector field effect in canvas element.

```js
vectorField({
    canvasId,
    canvasWidth = window.innerWidth,
    canvasHeight = window.innerHeight,
    maximumRotation = 10,
    density = 10,
});
```
